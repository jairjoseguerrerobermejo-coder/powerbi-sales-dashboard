# Dashboard de Análisis de Ventas – Power BI

## Descripción
Proyecto de análisis y visualización de datos enfocado en el estudio del comportamiento de ventas, permitiendo identificar tendencias, categorías más relevantes y patrones de consumo.

## Herramientas
- Power BI
- Excel

## Procesos realizados
- Extracción de datos desde diferentes fuentes
- Limpieza y transformación de datos (ETL)
- Modelado de datos relacional (tablas dimensión y hechos)
- Creación de medidas y KPIs
- Diseño de dashboards interactivos

## Análisis realizado
- Evolución de ventas por año y trimestre
- Comparación por categorías y subcategorías
- Análisis de precios unitarios
- Identificación de tendencias de ventas


## Resultado
Se desarrolló un dashboard interactivo que permite apoyar la toma de decisiones mediante el análisis visual de indicadores clave.
